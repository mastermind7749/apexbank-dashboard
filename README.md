# ApexBank Demo Dashboard

A modern, interactive banking dashboard demo built with **Tailwind CSS** and **Lucide Icons**.

## Features

✨ **Live Features:**
- Add custom transactions
- Real-time balance updates  
- Transaction history tracking
- Local storage persistence
- Beautiful dark theme UI
- Fully responsive design

## Demo

Visit: **https://mastermind7749.github.io/apexbank-dashboard**

## Technologies Used

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first styling (via CDN)
- **Lucide Icons** - Beautiful icon library
- **Vanilla JavaScript** - No dependencies required
- **LocalStorage API** - Client-side data persistence

## How to Use

1. Open the dashboard
2. Enter a transaction description (e.g., "Coffee", "Gas")
3. Enter the amount in dollars
4. Click "Execute Expense"
5. Watch your balance update and see the transaction in the history

All data is stored locally in your browser and persists across page reloads.

## Note

⚠️ **Demo Mode**: This is a demonstration dashboard. All data is simulated and stored locally. No real transactions are processed.
